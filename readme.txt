== Description ==
 
W4 Simple Ajax Search allows you to add a search field that searches through posts and page titles and immediately returns the resultsÂ 
 
== Installation ==
 

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php echo do_shortcode('[w4_sas_search_field]'); ?>` in your templates
